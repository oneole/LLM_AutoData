# 学生成绩分析平台测试文档

## 1. 简介

`app.py` 是一个使用 Streamlit 构建的学生成绩分析平台，旨在帮助用户通过自然语言查询和可视化分析来理解学生成绩数据。该平台连接到 MySQL 数据库，并提供以下主要功能：

*   **自然语言查询**: 用户可以使用自然语言描述查询需求，平台将其转换为 SQL 查询并执行。
*   **数据展示**: 平台以表格、图表和数据分析报告的形式展示查询结果。
*   **数据库内容预览**: 用户可以查看数据库中的表结构和数据。
